# ЛОГИКА ДЛЯ РЕАЛИЗАЦИИ

1.  **Константа PFX:** В самом начале скрипта определи константу с реальным значением `pfx`.

2.  **Логика Мобильного Off-Canvas Меню:**
    *   **Проверка:** ЕСЛИ в HTML существуют элементы `.${pfx}-burger-btn` и `#{pfx}-mobile-nav`, реализуй эту логику.
    *   **Функционал:**
        *   **A) Переключение состояния:** При клике на бургер, переключай класс `mobile-nav-is-open` на теге `<body>` и обновляй `aria-expanded` на кнопке. Это вызовет CSS-анимации.
        *   **B) Закрытие меню:** Закрывай меню при клике на ссылку внутри `#{pfx}-mobile-nav` или по нажатию `Escape`.
        *   **C) "Ловушка" фокуса (A11y):** Когда меню открыто (body имеет класс `mobile-nav-is-open`), фокус клавиатуры (Tab/Shift+Tab) не должен покидать пределы `#{pfx}-mobile-nav`.
        *   **D) Обработка Resize:** Автоматически закрывай меню, если окно браузера становится шире определенной точки (например, `800px`), чтобы избежать визуальных багов.

3.  **Логика Аккордеона (FAQ):**
    *   **Проверка:** ЕСЛИ в HTML существует блок `.${pfx}-accordion`, реализуй эту логику.
    *   **Функционал:**
        *   Найди все элементы `<details>` внутри аккордеона.
        *   Добавь обработчик события `click` на каждый `<summary>`.
        *   **Перехвати стандартное поведение:** Используй `event.preventDefault()` для `<summary>`, чтобы управлять открытием/закрытием вручную.
        *   **Анимация с помощью Web Animations API:**
            *   При открытии: сначала закрой все другие открытые `<details>` с анимацией, затем открой текущий с плавной анимацией высоты.
            *   При закрытии: анимируй высоту до нуля, и только после завершения анимации устанавливай атрибут `open = false`.
        *   Это создаст плавный и профессиональный эффект "один открыт за раз".

4.  **Логика для таблицы с горизонтальным скроллом:**
    *   **Проверка:** ЕСЛИ в HTML существует блок `.{pfx}-tblwrap` с атрибутом `data-scroll-hint="true"`.
    *   **Функционал:**
        *   Добавь обработчик события `scroll` на этот элемент.
        *   Как только пользователь начинает скроллить таблицу по горизонтали, удали атрибут `data-scroll-hint`, чтобы CSS мог скрыть подсказку о скролле.
