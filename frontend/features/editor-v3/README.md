# editor-v3

Каркас feature-модуля для рефакторинга редактора домена.

Структура:
- `components/` — UI-компоненты редактора (поэтапный вынос из page).
- `hooks/` — пользовательские хуки состояния/действий.
- `context/` — контексты редактора и AI-потоков.
- `services/` — константы, API-адаптеры, утилиты.
- `types/` — типы editor/AI для переиспользования.

В рамках шага R1.1 вынесены только базовые типы и константы без изменения поведения UI.
