# SSH Keys (Local Only)

Эта папка зарезервирована под локальные SSH-ключи для деплоя.

Рекомендуемые имена файлов:

- `media1_ed25519` — приватный ключ
- `media1_ed25519.pub` — публичный ключ
- `media2_ed25519` — приватный ключ (для следующего сервера)
- `media2_ed25519.pub` — публичный ключ

Файлы ключей не должны попадать в git.  
В репозитории хранится только эта инструкция и `.gitkeep`.

Пример ссылки из env:

- `DEPLOY_TARGETS=media1`
- `DEPLOY_MEDIA1_SSH_KEY_PATH=./secrets/ssh/media1_ed25519`
